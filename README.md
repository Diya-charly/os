CPU Scheduling Simulator

Overview

A web-based simulator for visualizing CPU scheduling algorithms:

FCFS (First Come First Serve)

SJF (Non-Preemptive)

The simulator generates a Gantt Chart and calculates:

  1.Waiting Time
  
  2.Turnaround Time
  
  3.Average Waiting Time
